## Usage

### Env Variables

Create a .env file in then root and add the following

```
REACT_APP_STRIPE_KEY=xxxx

REACT_APP_RAZOR_KEY=xxxx

REACT_APP_ENV=development
```

### Install Dependencies (frontend & backend)

```
npm install
```

### Run

```
# Run frontend (:3000)
npm start
```

## Build & Deploy

```
# Create frontend prod build
npm run build
```
